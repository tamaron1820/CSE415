# Backggamon with AI

BackggamonのゲームでAIの探索アルゴリズムを使って対戦させる

# Features

MiniMax法とAlphaBeta法を組み合わせて探索させる
Backgammonにおける評価関数を設定し、優位性を判断

```bash
pip3 install pygame
```
